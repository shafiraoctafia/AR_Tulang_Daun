# AR_Tulang_Daun

## Tujuan Project
Projek Ini bertujuan untuk membuat aplikasi pembelajaran interaktif yaitu pengenalan bentuk tulang daun untuk siswa sekolah dasar menggunakan teknologi Augmented Reality yang berisi :
- Halaman Utama
- Halaman Materi
- Halaman Informasi
- Halaman Panduan
- Halaman Pengaturan
- Halaman Main (Scan AR)
- Halaman Kuis
- Halaman Keluar
